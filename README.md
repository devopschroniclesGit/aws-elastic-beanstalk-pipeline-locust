# AWS SDLC Automation Project

This project demonstrates an end-to-end AWS-native CI/CD pipeline using:

- CodePipeline
- CodeBuild
- CodeDeploy
- Elastic Beanstalk
- Locust Load Testing
- DynamoDB
- CloudWatch Monitoring
